rboy
====

A Gameboy Emulator in Rust using SDL

Implemented
-----------

* CPU
  - All instructions correct
  - All timings correct
* GPU
* Keypad
* Timer
* MMU
  - MBC-less
  - MBC1
  - MBC3 (partial)
  - save games

Not implemented
---------------

* Serial
* Sound
